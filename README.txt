Ιδιαίτερα Μαθήματα Φυσικής — Static Site

Αυτή είναι στατική έκδοση για δωρεάν/απλό deploy στο Render ως Static Site.
Δεν χρειάζεται backend, Node.js ή environment variables.
Δοκιμαστική αλλαγή στο second-branch για εξάσκηση στο Git/GitHub.
Deploy στο Render:
1. New → Static Site
2. Build Command: άδειο
3. Publish Directory: .

Production domain:
https://www.idiaitera-fysikis.gr

Το canonical URL, το robots.txt και το sitemap.xml χρησιμοποιούν το production domain.
Η επικοινωνία παραμένει στατική μέσω email και τηλεφώνου.

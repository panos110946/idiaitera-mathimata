Ιδιαίτερα Μαθήματα Φυσικής — Static Site

Αυτή είναι στατική έκδοση για δωρεάν/απλό deploy στο Render ως Static Site.
Δεν χρειάζεται backend, Node.js ή environment variables.

Deploy στο Render:
1. New → Static Site
2. Build Command: άδειο
3. Publish Directory: .

Όταν αργότερα συνδέσεις κανονικό domain, τότε θα προστεθούν:
- canonical URL
- sitemap.xml με το τελικό domain
- πιθανό backend contact form, αν το θέλεις
